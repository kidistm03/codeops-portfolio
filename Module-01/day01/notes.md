# Day 01 Notes

## Five Terminal Commands

1. pwd - Shows the current directory.
2. ls - List the contents of the current directory.
3. cd - Changes the current directory.
4. mkdir - Creates a new folder.
5. rm - Removes a file.

## Three Git Commands

1. git init - Creates a new Git repository.
2. git status - Shows the status of the repository what's changed, staged, or ignored
3. git commit - Saves changes with a message.
